# Sidebar & Navigation

The sidebar helps you move around nBlick.

It gives you quick access to all main sections of the platform.

You can use it to switch between analyses, manage prompts, create content, and adjust settings.

## Top Section: Select Analysis

At the top of the sidebar, you will see: the name of the analysis you're currently on.

If you click on it, it's a dropdown menu that lets you:

- Switch between different analyses
- Create new analyses
- Manage existing ones

> 💡 Each analysis is separate. Data does not mix between analyses.

# Main Sections

The sidebar is divided into clear sections.

## General

These sections help you understand and manage your analysis.

### Overview

The **Overview** page shows:

- Visibility summary
- Key metrics (Sentiment, Share of Voice and Keywords)
- High-level insights

Use this page to quickly understand  your current AI visibility.

### Prompts

The **Prompts** section shows:

- All tracked prompts
- AI-generated answers
- Citations
- Competitor mentions

You can:

- View answers
- Pause prompts
- Delete prompts
- Track changes over time

Prompts are the foundation of AI visibility tracking.

### Configuration

The **Configuration** section allows you to consult:

- Analysis settings
- Categories
- Competitors
- Audience context

Use this section to remember the configuration of your analysis.

If you need to change any parameter we recommend creating a new analysis. 

## Agents

Agents are tools that help you take action.

### Content

The **Content** section allows you to:

- Generate AI-optimized articles
- Improve visibility for selected prompts
- Create long-form content based on categories

This is where you move from analysis to action.

## Settings

### Profile

The **Profile** section lets you:

- Update account details
- Manage personal information

# How to Use the Sidebar Effectively

Here is the recommended workflow:

1. Start in **Overview**
2. Go to **Prompts** to see detailed answers
3. Use **Content** to improve visibility
4. Adjust settings in **Configuration** if needed

# Best Practices

- Keep your analyses organized
- Use clear names
- Review prompts regularly
- Use Content generation to improve weak areas

# Summary

The sidebar gives you access to:

- Analysis selection
- Performance overview
- Prompt tracking
- Content generation
- Settings and admin controls

It is your main navigation tool inside nBlick.
