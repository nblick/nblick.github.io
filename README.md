# nBlick Documentation

Welcome to the nBlick documentation!

Project documentation built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/).

## Local Development

### Docker

```bash
docker compose up
```

### uv

```bash
uv run mkdocs serve
```

Site will be available at [http://localhost:8000](http://localhost:8000) with live-reload.

## Deployment

The site is automatically deployed to GitHub Pages on every push to `main` or `master` via GitHub Actions.
